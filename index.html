Get started with Synthetic User Testing

This skill lets Claude simulate how real users would experience your website or prototype. It tests through the eyes of three HR-tech personas and generates interactive reports with actionable improvement prompts.

1
Download the .skill file and install it in Claude — drag it into your conversation or use "Copy to your skills"
2
Tell Claude something like: Test https://example.com as Olivia — it will navigate the site, evaluate it through the persona's eyes, and generate an interactive HTML report with 7 tabs
3
Share your report by pushing the HTML file to the GitHub repo — it auto-deploys to this page

Publishing reports

After generating a report, push it to the shared GitHub repo. The fastest way is with the included publish.sh script:

./publish.sh path/to/report.html \ --site "Site Name" \ --date "2026-03-19" \ --personas "Olivia (HR Admin)" \ --task "Task description" \ --verdict "Short verdict" \ --verdict-type "mixed"

Or manually: copy the HTML file to reports/, add an entry to index.html, commit and push. Vercel auto-deploys within seconds.
